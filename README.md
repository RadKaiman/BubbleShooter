# BubbleShooter

## Описание
Игра-головоломка, цель которой лопать цветные пузыри, стреляя по ним группами из трех и более шаров одного цвета

## Геймплей
- Прицеливание: оттяните шарик назад. Появляется линия траектории
- Обычный выстрел: при неполном натяжении шарик прилипает к первому встреченному шару, образуя комбинации
- Идеальный выстрел (100% натяжение): шарик пробивает первый шар на пути, занимает его место. Линия траектории раздваивается
- Комбинации: если после прилипания образуется группа из 3+ шаров одного цвета то они лопаются
- Зависшие шары: шары, потерявшие связь с потолком, падают и лопаются
- Победа: в верхнем ряду остаётся менее 30% шаров от начального количества
- Поражение: закончились ходы

## Архитектура
- MVC: полное разделение данных, отображения и логики
- Вся баллистика рассчитана вручную без использования физики Unity и Raycast
- Object Pool шаров для оптимизация памяти и производительности
- UniTask для эффективной работы с асинхронностью
- Анимации через DOTween
- Сохранения через PlayerPrefs
